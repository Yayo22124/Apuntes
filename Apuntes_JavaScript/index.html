Fundamentos de JavaScript

Para imprimir resultados o mensajes con JavaScript es necesario usar:

  • 'console.log()': Muestra mensajes en la consola del navegador
  • 'alert()': Muestra ventanas emergentes para imprimir resultados.

Para crear variables se pueden usar:

  • 'let': Permite crear variables a las cuales se les puede modificar su valor.
  • 'const': Permite crear constantes las cuales solo admiten lectura, es decir, su valor no puede ser modificado.
  • 'var': Este tipo de variable tiene un scope (alcance) global, es decir, aunque se encuentre dentro de una funcón o unas {}, se puede acceder a esta variable desde fuera.

Las principales caracteristicas de JavaScript son:

  • Tipado Dinámico: No es necesario declarar tipos de datos, ya que este es asignado de forma automática al crear una variable y asignarle un valor.
  • Tipado Débil: Las variables pueden cambiar su tipo de dato sin necesidad de ser convertidos de forma manual, como pasar de String a Int.
